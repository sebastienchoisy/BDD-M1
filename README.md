# BDD-M1

Liste des méthodes de récupération de données à partir des fichiers : 

### CSV File Reader

**getPersonsDataListFromCsvFile:**  renvoit une liste d'objets Person à partir de person_0_0.csv.

**getFeedBacksDataListFromCsvFile:**  renvoit une liste d'objets Feedback à partir de Feedback.csv.

**getProductsFromCsvFile:**  renvoit une liste d'objets Product à partir de Product.csv.

**getBrandByProductFromCsvFile:**  renvoit une liste d'objets BrandByProduct à partir de BrandByProduct.csv.

**getTagsByPersonFromCsvFile** : renvoit une liste d'objets TagsByPerson à partir de person_hasInterest_tag_0_0.csv.

**getLinkByPersonFromCsvFile:** renvoit une liste d'objets LinkByPerson à partir de person_knows_person_0_0.csv.

**getPostsFromCsvFile:** renvoit une liste d'objets Post à partir de post_0_0.csv.

**getAuthorByPost:** renvoit une liste d'objets AuthorByPost à partir de post_hasCreator_person_0_0.csv.

**getTagsByPostFromCsvFile:** renvoit une liste d'objets TagsByPost à partir de post_hasTag_tag_0_0.csv.

**getVendorsFromCsvFile:** renvoit une liste d'objets Vendor à partir de Vendor.csv.

### XML File Reader

**getInvoicesFromXml:** renvoit une liste d'objects Invoice à partir de Invoice.xml
